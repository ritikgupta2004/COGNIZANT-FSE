# Bootstrap Structure and Files - Exercise 2.1

## Bootstrap Directory Structure

When Bootstrap is installed using npm, it is available inside:

node_modules/bootstrap/

### 1. css/

Purpose:
- Contains Bootstrap CSS files.
- Used to style HTML elements.
- Includes both normal (.css) and minified (.min.css) files.

Example:
- bootstrap.min.css

### 2. js/

Purpose:
- Contains Bootstrap JavaScript files.
- Used for interactive components such as:
  - Modal
  - Dropdown
  - Carousel
  - Tooltip
  - Collapse
  - Offcanvas

Example:
- bootstrap.bundle.min.js

The bundle version already includes Popper.js.

### 3. icons/

Purpose:
- Bootstrap Icons are a separate icon library.
- Provides SVG icons that can be used in buttons, navigation bars, forms, and other UI components.

Examples:
- house
- search
- person
- heart
